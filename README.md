# Анализ тональности комментариев в сообществе «Автомобильные истории» (VK)

Этот репозиторий содержит материалы курсовой работы по теме анализа тональности комментариев
в сообществе «Автомобильные истории» социальной сети ВКонтакте.  
Цель работы — построить конвейер от загрузки данных через VK API до моделирования и оценки
качества классификации тональности (negative / neutral / positive).

## Структура репозитория

- **VKapi.ipynb** — ноутбук для сбора данных из VK:
  - подключение к VK API по токену;
  - выгрузка постов и комментариев для указанного `group_id`;
  - сохранение результатов в CSV-файл `vk_comments_raw.csv`.

- **vk_comments_raw.csv** — сырые комментарии, полученные из VK API:
  - `post_id` — идентификатор поста;
  - `comment_id` — идентификатор комментария;
  - `from_id` — идентификатор пользователя;
  - `date` — время публикации (UNIX timestamp);
  - `text` — текст комментария.

- **kursovaya.ipynb** — основной ноутбук курсовой работы:
  - загрузка и предобработка комментариев;
  - автоматическая разметка тональности с помощью модели RuBERT;
  - описательная статистика (распределение классов, длина комментариев, частотный анализ);
  - извлечение признаков с помощью TF-IDF;
  - обучение моделей (MultinomialNB, Logistic Regression, LinearSVC);
  - сравнение качества и интерпретация результатов (матрица ошибок, важные признаки).
